# PL

Założenie ninieżej aplikacji będzie umożliwoało przechowywanie transakcji pieniężnych danego użytkownika. Dodatkowo aplikcjai ma umożłiwać gerewoanie zestawień transakcji w ujęciu: 
* dziennym,
* tygodniowym,
* miesięcznym,
* kawrtalnym,
* rocznym.
